Form
Fields with label for:
 [X] Fullname
 [X] Phone
 [X] Email
 [X] Address
 [X] DNI/NIF/NIE with regexp
 [X] Multi-line box to write long comments

All the inputs of the form must use their own type to ensure that the data entered is correct.

CSS styles:
 [] Select all the H2 elements (using one instruction) and change at least the color property
 [] Create a class that applies a border property and use in it in one element
 [] Create a class that applies margin and padding properties
 [] Create a ul or ol element and change the list-style
 [] Select all the paragraphs that are child elements of a div and change the font-weight to bold
 [] Select all the images that have a alt attribute (with one instruction) and change their width properties
 [] Change the background color and text color of all the form elements
 [] Change the document background (color or image)
 [] Create multiple classes to define all the font sizes in your site
 [] Implement some CSS3 rules that bring something new or creative to your site
 
 [] You cannot use third party libraries for the development of this pill
 [] You must take screenshots of the main browsers to ensure that the content is displayed correctly
 [] You can create your own styles in another CSS file that is different from the one mentioned in the point 1.3
 [] Both the code and the comments must be written in English
 [] The code of the different programming languages that you will use should be stored in a different file with the 
   appropriate extension (html => file.html, css => main.css). You should not mix html and CSS code in the same file.
